# Field Survey & Inspection App

A modern, enterprise-grade mobile application built with React Native and Expo for managing field surveys and site inspections.

## Features

- **Global State Management:** Seamlessly syncs data across screens using React Context.
- **Dynamic Dashboard:** View statistics for total surveys and high-priority tasks in real-time.
- **Survey Management:** Create new surveys and view history dynamically.
- **Custom Drawer Navigation:** A sleek, custom-designed sidebar drawer with an indigo gradient header and custom active/inactive tab states.
- **Profile & Settings:** Update your employee profile and adjust application preferences.

## Tech Stack

- **Framework:** React Native (Expo)
- **Routing:** Expo Router
- **State Management:** React Context API
- **Styling:** Custom StyleSheet (Vanilla RN) & `expo-linear-gradient`
- **Icons:** Ionicons (`@expo/vector-icons`)

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository (if applicable) and navigate to the project folder:
   ```bash
   cd survey_inspection_project
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the Expo development server:
   ```bash
   npm start
   ```

### Running the App
- Press `i` to open in an iOS simulator.
- Press `a` to open in an Android emulator.
- Press `w` to open in a web browser.
- Alternatively, scan the QR code using the **Expo Go** app on your physical device.
